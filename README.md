# BOOK RECOMMENDATION WEBSITE USING LANGCHAIN

## setup

1. Make sure your folder structure looks like this:

```
/project
  app.py
  .env
  /templates
     index.html
  /static
     style.css
     script.js
```

2. Install requirements if you haven't: pip install flask langchain-core langchain-openai pydantic python-dotenv

3. Run the app: python app.py

4. Open http://127.0.0.1:5000 in your browser.
